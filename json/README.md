# JSON 整形ツール

## 概要
見づらい、あるいは1行に圧縮された（Minifyされた）JSON文字列を、適切なインデントを付与して人間が読みやすい形式にフォーマット（整形）するツールです。

## 主な機能
1. **リアルタイム整形**: 左側のテキストエリアにJSONを貼り付けると、右側に整形済みのJSONが瞬時に表示されます。処理はすべてブラウザ上（クライアントサイド）で行われます。
2. **インデント設定の変更**: 整形時のインデントを「2スペース」「4スペース」「タブ(\t)」から選択可能です。設定を変更すると即座に出力結果が再描画されます。
3. **構文エラーのリアルタイム検知**: 入力されたテキストが有効なJSONフォーマットでない場合、右側の出力は停止し、上部に赤いエラーメッセージ（パースエラーの詳細）を表示します。
4. **ワンクリックコピー機能**: フォーマットされた結果を1クリックでクリップボードへコピーします。

## 処理の仕様 (JavaScript)

- JS標準の `JSON.parse()` メソッドを用いて入力テキストの構文解析を実行します。このとき `try-catch` ブロックを使用し、パース失敗時のエラーハンドリング（`catch(e)`）によりエラーメッセージを抽出・表示します。
- パースに成功したJSONオブジェクトは、`JSON.stringify(jsonObject, null, indentSpace)` メソッドを用いて、ユーザーが選択したインデント設定を適用して文字列化し、出力テキストエリアの `value` にセットします。
- ダークモード非対応のJSONパーサーライブラリ等への依存を無くし、Vanilla JavaScriptで軽量に実装されています。

## 利用シナリオ
- バックエンドAPIから取得した1行の生JSONデータの可読性向上。
- 手書きのJSON設定ファイルの構文チェック（カンマの抜け漏れなどの発見）。
- JSONデータのデバッグやドキュメントへの貼り付け用フォーマッティング。
