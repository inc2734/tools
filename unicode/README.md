# Unicode 文字変換ツール

## 概要
Unicodeエスケープシーケンス（`\uXXXX`）で表現された文字列を、人間が読める通常の文字列（平文）にリアルタイムでデコードするブラウザベースのツールです。

## 主な機能
1. **リアルタイムデコード**: 左側のテキストエリアに入力した瞬間、JavaScriptにより右側のテキストエリアへ即座に変換結果が反映されます。サーバーとの通信は発生しません。
2. **ワンクリックコピー機能**: 「コピー」ボタンをクリックするだけで、出力結果をシステムのクリップボードにコピーできます。
3. **入力クリア**: 「クリア」ボタンで入力及び出力欄を一括で空にします。

## 処理の仕様 (JavaScript)

- バックスラッシュ(`\`) と `u` に続く4桁または5桁の16進数を正規表現 `/\\u([\d\w]{4,5})/gi` で検知します。
- JSの `String.fromCharCode(parseInt(match, 16))` を用いて文字コードから実文字へ変換を行います。
- 不正なシーケンスや変換不能な文字が含まれる場合は、フォールバックとして元の文字列（`\uXXXX`のまま）を維持して出力します。
- ダークモード対応: OS側で設定されているダークモード設定を読み取り、UIテーマを動的に適用します（状態は `localStorage` に保持されます）。

## 利用シナリオ
- APIからの意図しないUnicodeエスケープ付きJSONレスポンスの解読。
- ソースコード内に難読化や文字化け防止のためにエスケープされた日本語等の文字列の確認。
