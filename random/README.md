# ランダム文字列生成ツール

## 概要
指定された長さや文字の組み合わせに基づいて、安全なランダム文字列をブラウザ内で生成するツールです。パスワードやトークン、APIキーなどの作成に利用できます。

## 主な機能
1. **カスタマイズ可能な生成条件**:
   - 文字数指定機能 (スライダーまたは数値入力、デフォルト: 12文字)
   - 使用する文字グループの選択 (英小文字、英大文字、数字、記号)
2. **リアルタイム生成**: 設定を変更するか「再生成」ボタンをクリックすると即座に新しい文字列が画面に表示されます。処理はすべてクライアントサイドで完結しています。
3. **ワンクリックコピー**: 生成された文字列を1クリックでシステムのクリップボードにコピーできます。

## 処理の仕様 (JavaScript)
- `window.crypto.getRandomValues()` を利用した暗号学的に安全な擬似乱数生成を行っています。`Math.random()` は使用していません。
- ユーザーが選択した文字種の組み合わせ（プール）から、指定された長さの分だけランダムに文字を抽出して結合します。
- UIの構築には Tailwind CSS v4 を使用し、モダンなデザインを採用しています。
- ダークモード対応: OS側で設定されているダークモード設定に自動で追従します。

## 利用シナリオ
- セキュアなパスワードの新規作成。
- アプリケーション開発時のランダムなダミーデータの生成。
- 一時的なアクセストークンやシークレットキーの発行。
