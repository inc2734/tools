# Tools

開発作業をサポートするためのブラウザ完結型のオンラインツール集です。データはサーバーに送信されず、すべてブラウザ内で安全かつ高速に処理されます。

## 概要

このプロジェクトは、フロントエンド開発者やWebクリエイター向けに、日常的なちょっとしたデータ変換作業を快適に行えるツールを提供することを目的としています。
UIにはTailwind CSS v4を使用したモダンなグラスモーフィズムデザインを採用し、OSの設定に連動するダークモードにも対応しています。

## 構成技術

- **フロントエンドビルド**: Vite (v7.3.1)
- **スタイリング framework**: Tailwind CSS (v4.0.0)
- **スクリプト**: Vanilla JavaScript
- **デプロイ**: GitHub Pages (GitHub Actions `deploy.yml` 経由)

## 含まれるツール

各ツールの詳細な仕様については、それぞれのディレクトリ内の `README.md` を参照してください。

- **[Unicode 文字変換 (unicode/)](./unicode/README.md)**: `\uXXXX` 形式のUnicodeエスケープシーケンスを通常の文字にデコードして表示します。
- **[JSON 整形 (json/)](./json/README.md)**: 未フォーマットのJSONデータを読みやすく整形インデントし、構文エラーを検証します。

## ローカルでの開発・実行手順

このプロジェクトはNode.js環境（v20.19以上、またはv22.12以上推奨）で動作します。

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動 (localhost:5173 などでプレビュー)
npm run dev

# 本番用のビルド生成 (dist/ フォルダへ出力)
npm run build
```

## 自動デプロイと運用

`main` ブランチにコードがプッシュされると、GitHub Actions が自動的にビルド (`npm run build`) を実行し、生成された `dist/` ディレクトリ内のファイルを **GitHub Pages** へデプロイします。運用時に手動でのビルドアップロードは不要です。
